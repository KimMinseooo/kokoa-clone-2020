### Display Position Summary

- Block : 한줄에 하나만 위치
- inline-block : 한줄에 여러개 위치
- inline : 안에 존재하는 콘텐츠의 양에 따라 크기가 가변

- relative : 원래 있어야 하는 위치에서 상대적으로 이동
- absolute : 아이템이 담겨있는 상자 안에서 위치가 변함
- fixed : 상자를 벗어나 윈도우 내부에서 위치가 변함
